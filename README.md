# Portafolio web para estudiante de diseño

Estructura:

```txt
portafolio-diseno/
├─ index.html
├─ styles.css
├─ script.js
└─ assets/
   ├─ hero-placeholder.svg
   ├─ project-branding.svg
   ├─ project-illustration.svg
   └─ project-ui.svg
```

## Qué editar

Busca los comentarios `CAMBIAR:` dentro de `index.html`, `styles.css` y `script.js`.

Reemplaza:
- Nombre.
- Especialidad.
- Frase del hero.
- Textos de Sobre mí.
- Imágenes de proyectos dentro de `assets/`.
- Links de Behance, LinkedIn, Instagram y correo.
- Links de cada proyecto.

## Publicar en GitHub Pages

1. Sube estos archivos al repositorio.
2. Asegúrate de que `index.html` esté en la raíz del repo.
3. Ve a Settings > Pages.
4. En Branch selecciona `main` y carpeta `/root`.
5. Guarda y espera a que GitHub genere el link.
